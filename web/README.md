## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
cd web
npm install
```

### Development

```bash
npm run dev
```

The app will open at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Usage

- Click **"+ New Entry"** to create a new journal entry
- Select entries from the sidebar to view/edit them
- Type your title and content in the editor
- Click **Save** or press **Ctrl+S** to save changes
- All entries are stored locally in your browser

## Design

The interface features:
- Dark background (#1a1a2e)
- Soft off-white text (#e9e9f0)
- Purple (#7c3aed) and blue (#3b82f6) accents
- Rounded corners and smooth spacing
- Calm, distraction-free writing environment
